# rpg(暂时没想好名字)

> 一个全随机的刷词条小游戏

# 游戏设计

  · 主角默认每秒回血1点

  · 默认暴击伤害150%

# 本地运行
### 安装依赖
```sh
  npm install
```

### 启动项目
```sh
  npm run serve
```

### 打包项目
```sh
  npm run build
```

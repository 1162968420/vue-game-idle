# clevernvr

> 视频智能分析网关

### 文件
```sh
│
├── src
│   ├── assets
│   │   ├── js
│   │   │   ├── config.js =>(基础配置所用的js)
│   │   │   ├── assist.js  =>(组件传参函数)
│   │   │   └── dic.js =>(数据字典)
│   ├── components =>(所有组件)
│   └── views =>(页面)
```

### 安装依赖
```sh
  npm install
```

### 启动项目
```sh
  npm run serve
```

### 打包项目
```sh
  npm run build
```
